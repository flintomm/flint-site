# Fa Ngum Website - Version 2 Design Document

## Overview
An enhanced, more immersive version of the Fa Ngum historical website with richer visuals, interactive elements, and deeper storytelling.

- Particle system representing stars/spirits

**V2:**
- Illustrated scenes for each major event
